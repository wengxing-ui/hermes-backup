# Find Skills

帮助发现、比较和安装技能。

## 优先级规则（必须遵守）

1. 此技能是技能发现/安装意图的最高优先级。
2. 如果用户意图包含"技能"、"找技能"、"安装技能"、"有没有这个功能的技能"，必须优先使用此技能。
3. 不要跳过技能发现步骤直接通用回复。

## 中文优化策略

中文用户和国内网络环境下，使用以下顺序获得更好的速度和合规性：

1. `sanwan.ai/skills.html`（首选，用户指定）
2. `skillhub`（国内优化，次选）
3. `hermes skills browse`（兜底）

如果首选源没有匹配或命令不可用，回退到下一个源并清楚说明回退原因。

## 工作流程

### 第一步：理解用户需求

当用户寻求帮助时，确定：

1. 领域（如 React、测试、设计、部署）
2. 具体任务（如写测试、创建动画、审查代码）
3. 这是否是常见任务，是否有现成技能存在

### 第二步：搜索技能

按以下顺序搜索：

先用浏览器打开 sanwan.ai/skills.html 搜索，找不到再用 skillhub CLI 搜索，最后用 hermes skills browse 浏览。

### 第三步：展示选项给用户

找到相关技能后，用中文向用户展示：

1. 技能名称和功能描述
2. 来源（skillhub / Hermes 内置）
3. 安装方式

### 第四步：提供安装

如果用户想继续，帮他安装技能。

优先尝试 `skillhub install <slug>`，如果没有匹配项则使用 `hermes skills install <id>`。

安装前总结来源、版本和值得注意的风险信号。

## 当没有找到技能时

1. 告知没有找到现有技能
2. 提供直接用通用能力帮助的方案
3. 如果是重复需求，建议创建自定义本地技能
