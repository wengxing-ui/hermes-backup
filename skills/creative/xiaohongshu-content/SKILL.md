# 小红书内容创作

小红书短视频脚本拆分和笔记排版美化。

## 视频脚本拆分规则

当用户提供长文案要求拆分为小红书视频时：

### 格式要求

每条视频包含以下要素：
- **封面**：独立吸引人的标题，一行即可
- **时长标注**：约X分X秒
- **正文**：不改变原文案内容，只做分段
- **结尾悬念**：第一条和第二条结尾留钩子引导看下一集，最后一条结尾带行动号召

### 拆分逻辑

按内容自然断点拆分，通常三集结构：
1. 第一集：开场钩子 + 第一个核心观点
2. 第二集：承上启下 + 第二个核心观点
3. 第三集：全局观点 + 行动号召收尾

### 格式示例

```
视频一：标题（约X分X秒）
封面：封面文字
正文内容...
下期讲XXX，更严重👇

视频二：标题（约X分X秒）
封面：封面文字
接上期。正文内容...
还有一个风险，更隐蔽。

视频三：标题（约X分X秒）
封面：封面文字
正文内容...
结尾行动号召
```

## 笔记排版规则

将长文案转化为小红书笔记时：

- 用 emoji 做视觉分隔符（🇺🇸 ⚠️ ❌ 💡 🔑 📌 💬）
- 每段不超过三行，善用换行
- 重点信息用符号突出（▪️ ❶ ❷ ③）
- 关键词用「」标注
- 开头有吸引力，结尾有互动引导
- 标签用 # 放在末尾

## 注意事项

- 绝对不改变用户提供的原文案内容
- 只做分段和排版美化
- 保持原文的语言风格和用词
- 拆分后的每段都要独立完整，有头有尾
